## Go by Example 中文版
Go 是一门被设计用来构建简单、高效、可信赖软件的开源程序设计语言。

Go by Example 是对 Go 基于实践的介绍，包含一系列带有标注说明的示例程序。查看第一个例子或者浏览下面的完整列表吧。
